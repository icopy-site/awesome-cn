## Platforms

- [Hook.io](https://hook.io) - 微服务和webhook托管的开源提供商.
-  [KintoHub！[c]]（https://www.kintohub.com） - 微服务包管理器使开发人员能够轻松构建和共享云就绪功能.

- [Netflix OSS](https://netflix.github.io/) -  Netflix开源软件生态系统.
- [OpenWhisk](http://openwhisk.org/) - 无服务器，开源云平台，可响应任何规模的事件执行功能.

-  [VAMP！[c]]（http://vamp.io/） - 轻松，轻松地构建，部署和管理微服务.

## Frameworks / Runtimes

- [Quasar](https://github.com/puniverse/quasar) -  JVM的光纤，通道和演员.
- [ScaleCube](https://github.com/scalecube/scalecube) - 用于为JVM构建反应式微服务的工具包：低延迟，高吞吐量，可扩展和弹性.

- [Vert.X Toolbox](https://github.com/vert-x3/vertx-microservices-toolbox) - 一组Vert.x组件，用于构建反应式微服务应用程序.
- [Wangle](https://github.com/facebook/wangle) - 一个框架，以一致，模块化和可组合的方式提供一组公共客户端/服务器抽象，用于构建服务.
- [WildFly Swarm](http://wildfly-swarm.io/) - 以足够的Java EE运行时开始的微服务，基于 [Eclipse MicroProfile](https://projects.eclipse.org/proposals/eclipse-microprofile).

- [Microservices Resource Guide](http://martinfowler.com/microservices/) -  Martin Fowler选择的文章，视频，书籍和播客可以教你更多关于微服务架构风格的内容.
- [Microservice Patterns](http://microservices.io/) - 微服务架构模式和最佳实践.
- [Microservice Antipatterns and Pitfalls](https://www.oreilly.com/ideas/microservices-antipatterns-and-pitfalls) - 微服务主要是反模式和陷阱.
- [The 12-Factors App](http://12factor.net) - 构建软件即服务应用程序的方法.

## Emerging Technologies

- [BigchainDB](https://www.bigchaindb.com/) - 可扩展的区块链数据库.
- [Blocknet](http://blocknet.co/) -  Blocknet可以通过基于区块链的P2P网络架构提供微服务.
- [Blockstack](https://blockstack.org/) - 一个新的分散式互联网，用户可以在其设备上控制数据和应用程序.
- [CEPTR](http://ceptr.org/) - 用于分布式社交计算的可重构介质，语义自描述协议栈，以及用于仿生计算的分形相干上下文的创建者.
- [Edgware Fabric](http://edgware-fabric.org/) - 轻巧，灵活的服务总线，适用于物理世界中网络边缘的系统.
- [Hyperledger](https://www.hyperledger.org/)   - 为促进跨行业区块链技术而开发的开源协作工作.  它是由Linux基金会主办的全球合作，包括金融，银行，物联网，供应链，制造和技术领域的领导者.
- [Luna](http://www.luna-lang.org/) - 视觉和文本功能编程语言，侧重于生产力，协作和开发人体工程学.
- [MultiChain](http://www.multichain.com/) - 用于构建区块链的开放平台.
- [Node-RED](http://nodered.org/) - 以新颖有趣的方式将硬件设备，API和在线服务连接在一起的可视化工具.
- [ops](https://ops.city) - 免费的开源工具，用于构建，运行和部署现有的Linux应用程序作为unikernel.
- [Pony](http://www.ponylang.org/) - 开源，面向对象，演员模型，功能安全，高性能的编程语言.
