    Awesome Symfony ¶

    很棒的列表 Symfony 捆绑包、实用程序和资源.

    目录：

    Administration¶

    • AdminCrudBundle - AdminCrudBundle - 使用 SensioGeneratorBundle. 扩展控制器、添加分页器、过滤器等.
    • AdmingeneratorGeneratorBundle - Symfony2 的 Admingenerator，解析 generator.yml 文件以构建类
    • EasyAdminBundle - Symfony 应用程序的简单管理生成器
    • SonataAdminBundle - AdminBundle - 缺失的 Symfony2 管理生成器
    • AdminLTEBundle - 基于 AdminLTE 模板的管理主题
    • Umbrella framework - 用于创建管理后端的管理组件和主题.
    • TablerBundle - 基于 Tabler 模板的管理主题. 它附带了许多树枝助手（函数、过滤器、嵌入、宏和包含）.

    Development¶

    • AccessibleBundle - 使用强大的注释定义类的 getter、setter 和构造函数.
    • ApiExceptionBundle - 异常 API 包.
    • AvAjaxBundle - 这个包提供了一个简单的结构来运行 ajax 操作.
    • BeelabTestBundle - 这个包只包含 Symfony WebTestCase 的自以为是的扩展.
    • CacheAdministrationBundle - 这个包包括一个控制器，带有清除各种缓存的操作.
    • ControllerExtraBundle - Set of useful Controller annotations.
    • DunglasActionBundle - 基于 Action-Domain-Responder 模式的 Symfony 控制器系统的替代品.
    • http-bundle - 提供额外的 HTTP 相关功能.
    • JMSDebuggingBundle - 提供高级调试工具.
    • LadybugBundle - 简单且可扩展的 PHP 转储程序.
    • LiipCodeBundle - 一组 Symfony2 控制台命令，帮助开发人员处理识别类、模板、捆绑包、服务等的各种方式.
    • LiipFunctionalTestBundle - 一些用于在 Symfony 2 中编写功能测试的辅助类.
    • ListenersDebugCommandBundle - 用于调试侦听器的控制台命令.
    • ParamConverterBundle - 这个包为 Symfony 提供额外的参数转换器.
    • PhpMetricsCollectorBundle - 在 Symfony2 调试工具栏中集成 PhpMetrics.
    • PsyshBundle - 提供加载了 Symfony 上下文的增强型 PHP 交互式 shell.
    • PUGXGeneratorBundle - 增强了 SensioGeneratorBundle.
    • redaktilo-bundle - 将编辑器集成到 Symfony 2 中.
    • SandboxBundle - 在沙盒环境中覆盖控制器逻辑和响应.
    • StatsDClientBundle - 使用随时可用的统计信息轻松监控您的 symfony2 生产环境.
    • TagDebugCommandBundle - 集成 TagDebug 库以检查和调试标签.
    • TwigReflectionBundle - 显示 Twig 中的内容.
    • WebfactoryExceptionsBundle - 轻松开发定制的、用户友好的错误页面.
    • WebProfilerExtraBundle - 在网络分析器中添加路由、容器、资产和树枝信息.
    • XhprofBundle -XHProf 包.

    Forms¶

    • KarserRecaptcha3Bundle - 将 Google ReCAPTCHA v3 集成到 Symfony 中.
    • LexikFormFilterBundle - Lexik 表单过滤器包.

    Internationalization¶

    • BazingaJsTranslationBundle - 一种向客户端应用程序公开 Symfony2 翻译消息的好方法.
    • JMSI18nRoutingBundle - 捆绑多语言网站以支持国际航线.
    • JMSTranslationBundle - 轻松翻译您的网站 - 提取消息并通过基于网络的 UI 进行翻译.
    • LuneticsLocaleBundle - 从不同的参数猜测访问者的语言环境.
    • TimezoneBundle - 服务器端时区检测.
    • TranslationFormBundle - 翻译领域，以促进可翻译原则扩展的使用.

    Miscellaneous¶

    • AnhTaggableBundle - Bundle 提供 doctrine-extensions-taggable 的集成，添加用于编辑标签和标记的表单类型.
    • AntennaBundle - 更容易集成 Antenna（并通过它 JWT）身份验证.
    • APYBreadcrumbTrailBundle - 此捆绑包提供注释和 PHP 方法来生成碎片痕迹.
    • APYDataGridBundle - 数据网格包.
    • AvAlertifyBundle - 此捆绑包可让您轻松协调警报和其他通知.
    • AviaryBundle - 这个包提供了多个文件上传，基于 BlueImp jQuery 文件上传包. 它还提供 Aviary 编辑图像功能.
    • AvListBundle - 在 Symfony2 中轻松制作分页和可排序列表.

    • BazingaFakerBundle - 将很棒的 Faker 库放入 Symfony2 DIC 并用假数据填充你的数据库.
    • BazingaGeocoderBundle - 集成 Geocoder 库.
    • BCCMyrrixBundle - Myrrix 是基于 Apache Mahout 库构建的推荐引擎.
    • BGBarcodeBundle - 使用我们的条码生成器基础库进行条码渲染的捆绑包.
    • BlogBundle - 简单的博客模块.

    • BreadcrumbsBundle - 一个小面包屑包.
    • BrowscapBundle - 捆绑访问 browscap 信息.

    • CheckBundles - 检查已安装但未在 AppKernel 捆绑包中激活.
    • CloudBackupBundle - 能够备份您的数据库并将其上传到云端.
    • ConsoleBundle - 浏览器中的命令行界面.

    • FlorianvSwapBundle - 这个 Bundle 集成了 Swap 库.
    • FMBbCodeBundle - 为 Symfony 项目添加 BBCode 支持.
