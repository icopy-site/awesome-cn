    Awesome Web Apps (that work without login)¶

    精选的优秀 Web 应用程序列表，无需您创建帐户即可运行.

    Inspiration¶

    拯救世界免于为每件该死的东西创建用户帐户和安装软件应用程序.

    Some pointers¶

    • 此列表中链接的网络应用程序无需登录即可提供大部分核心功能. 由于缺乏合适的替代方案，一些需要注册的网站也已包含在此列表中. 他们在列表中被标记为“[Account]”.
    • 这里提到的大多数应用程序都提供某种用户帐户功能来保存历史记录和进度. 但如前所述，使用这些应用程序的主要功能不需要用户帐户.

    Audio and Video¶

    • bfxr - 为您的游戏制作音效.
    • MP3 Cut - 在线音频切割器.
    • TwistedWeb Online - 强大的音频编辑器. 文件未在线保存.
    • Filelab Audio Editor - 具有修剪、合并、剪切音频等基本功能的音频编辑器.
    • Until AM - 在线音乐混音器、DJ 软件.
    • YOU.DJ - 在线音乐混音器，DJ 软件从 SoundCloud 采购音乐.
    • VideoToolbox [Account] - 具有所有常见编辑功能的视频编辑器. 大小限制为 600mb.
    • AI Vocal Remover - 通过人工智能将声音与音乐分开.
    • Ambient Mixer - 聆听免费的音频氛围（例如苏格兰雨/斯莱特林公共休息室）或在线混合您自己的环境声音.
    • Vileo - 录制您的屏幕或网络摄像头并从您的浏览器中下载视频.
    • Youtube Dynamic Playlists - 创建 YouTube 视频的即时播放列表.

    Business and Finance¶

    • ShapeShift.io - 即时加密货币转换.
    • Cryptrader.com - 加密货币价格图表和指标. 保存您的设置需要一个帐户.
    • TradingView.com - 来自各个交易所的实时信息和市场见解. 需要一个帐户来保存设置.
    • ICOStats.com - 跟踪和比较 ICO 的性能. 显示详细的统计数据，如自 ICO 以来的投资回报率、自 ICO 以来的投资回报率与 ETH，以及用于比较 ICO 历史表现的图表.
    • InvoiceToMe - 使用您公司的详细信息从各种模板生成专业发票.

    Communication¶

    • Tlk.io - 使用您想要的任何 URL（例如 tlk.io/some-room）创建一个房间并分组聊天.
    • Discord - 在线语音和文本会议. 开始时会询问用户名以创建临时头像.
    • Gruveo - 在线视频会议. 就像 Tlk.io，易于记忆的自定义 URL.
    • Whereby - 在线视频会议. 创建任何房间，例如 whereby.com/some-room 并开始聊天.
    • Videolink2 - 在线视频会议. 链接已生成，分享链接以邀请其他参与者.
    • GetARoom.io - 在线视频会议. 具有移动浏览器支持，只需单击开始新呼叫并共享链接.
    • FaxZero - 免费发送在线传真.
    • A2fax - 在线发送免费传真. 封面上没有广告. 将您自己的徽标添加到封面.
    • GoTeam - 视频会议室. 创建任何房间，如 goteam.video/some-room . 创建房间后询问用户名.

    Document Editors¶

    Latex¶

    • Papeeria - Latex 预览，带有无需注册即可下载为 PDF 的选项.
    • Overleaf [Account] - 无限的协作者和项目，免费帐户中的 git 访问权限.
    • Sharelatex [Account] - 具有无限私人项目的 Latex 编辑器. 免费帐户中有 1 位合作者.

    Markdown¶

    • Dillinger - 强大的 Markdown 所见即所得，可导出为 HTML 和 PDF. 用于保存文档的用户帐户.
    • Github.md - 具有打印功能的简单 GitHub 风格的 Markdown (GFM) 预览.
    • HackMD - 具有预览功能的协作、强大、实时在线降价编辑器.
    • Markdown Live Preview - 即时降价预览.
    • Markdownify - 最小的 Markdown 编辑器.
    • StackEdit.io - 丰富的浏览器内降价编辑器，可离线工作.

    Others¶

    • EtherCalc - 在线电子表格编辑器，导出到 Excel.
    • PdfEscape - Edit or create PDFs in browser itself.
    • Browserpad - 浏览器中的无服务器纯文本编辑器. 允许您打开和保存纯文本文件.
    • WriteURL - 协作式实时在线文本编辑器.

    Drawing¶

    • SVG-Edit - 具有许多功能的强大 SVG 编辑器. 开源.
    • Method Draw - 简洁且功能丰富的 SVG 编辑器. 出口到 PNG.
    • A Web Whiteboard - 触摸友好的在线协作白板应用程序.
    • Witeboard - 为您的团队准备的实时白板. 无需注册.
    • Draw.io - 流程图和 UML 绘图工具. 有很多绘图样式可供选择. 导出为 PNG 并与 Google 云端硬盘集成.
    • AutoDraw - 借助 Google 的机器学习，绘制精美的草图. 图纸可以下载为 PNG.
    • kleki - 浏览器中类似 MS Paint 的界面.
    • JSPaint - 浏览器中 1990 年代 MS Paint 的复制品. 开源.
    • bubbl - 创建思维导图，在浏览器中集思广益.
    • ScribbleMaps - 创建自定义谷歌地图.
    • Sketchpad - 创建数字艺术作品并导出为流行的图像格式 JPEG、PNG、SVG 和 PDF.
    • Piskel - Free online editor for animated sprites & pixel art
    • PixelCraft - 像素艺术编辑器和动画创作工具

    File Converters¶

    • Cloud Convert - 从任何东西转换成任何东西，包括音频、视频、字体、文档等等. 与 Google Drive 和 Dropbox 等云存储服务完美集成. 直接链接.
    • Online Convert - 音频、视频、图像、文档、电子书和存档转换器的集合. 转换后没有文件大小限制和直接下载链接.
    • PDF <=> EPUB - PDF 到 EPUB 转换器，反之亦然.
    • Pandoc Try - 使用 Pandoc 的内部标记转换. 支持 - Latex、Markdown、HTML、RST 等等.
    • Print Friendly - 将任何网页转换为简单格式. 然后下载或打印为 PDF.
    • OnlineOCR - 将扫描的 PDF 转换为 DOC、XLS 或 TXT.
    • GifDeck - 将幻灯片从 slideshare 转换为 GIF.
    • favicon-generator - 通过上传您想要的图像，为您的网络应用程序生成图标或为您的 Android 或 iOS 应用程序生成图标.
    • freetools.site - 免费在线工具. 转换或编辑文档、图像、音频、视频等.

    File Hosting/Sharing¶

    • EFShare - 点对点安全文件共享.
    • RGhost - 文件托管服务，最大限制 100mb 无需登录.
    • ImgSafe - 带有小型直接链接的图像托管服务.
    • Clyp - 无限制的音频共享，丰富的API.
    • SendVid - 视频托管服务.
    • Archive.org [Account] - 任何类型的无限文件托管，对带宽和上传大小没有限制.
    • MultCloud - 在多个云服务之间管理、移动、复制和迁移数据的云服务. 支持所有主要的云服务. 无需注册，2TB云存储，大文件直接下载到云端，无大小限制.
    • Firefox Send - Mozilla 的免费文件共享程序. 最多 1gb 没有帐户，2.5gb 有帐户.

    Games¶

    • Typeracer - 在与他人比赛时评估您的打字速度.
    • Spark Chess - 根据您的技能水平，学习下棋、与全球用户一起玩或与 AI 一起玩.
    • Miniclip - 收集各种免费游戏，无需提供您的电子邮件即可玩. 需要闪光灯.
    • CrossCode - 一款以遥远的未来为背景的复古风格 2D 游戏. 带有出色的游戏机制，例如连击、拼图、技能树、任务、物品等.
    • agar.io - 目标是通过吃比玩家细胞小的琼脂和细胞来获得尽可能多的质量，同时避免吃比玩家细胞大的琼脂和细胞.
    • paper-io - 游戏的目标是控制 100% 的竞技场. 您将扮演一个不断移动的小方块，并有一个与您的方块颜色相对应的小起始区域.
    • diep.io - Diepio 是一款你扮演坦克并射击东西的游戏. 射击形状以升级您的坦克，然后射击玩家以获得大量积分.
    • skribbl.io - 几个玩家连接到一个大厅，其中一个玩家得到一个词，他们需要画一个词，但不说是什么，而其他玩家必须识别它.
    • surviv.io - Surviv.io 是一款 2D 大逃杀游戏. 每场比赛你只有一个活人，没有重生.
    • nitrome - 收集免费的像素艺术游戏. 新游戏不需要 Flash.
    • Orion - 棋盘/益智游戏. 巧妙地组合袋中的方块以填满棋盘.
    • Gidd.io - 经典游戏合集，如 UNO、Yatzy、Scattergories 和 GeoGuess.

    Graphics, Image and Design¶

    • PhotoPea - 看起来像 Photoshop 的图像编辑器，具有它的大部分功能.
    • Pixlr Editor - Pixlr 的图像编辑器. 很多功能.
    • Photo Editor | Befunky - 具有大量过滤器和效果的图像编辑器.
    • Logo Makr - 带有精美 SVG 集合的徽标制作工具. 需要付费才能将徽标导出为 SVG（PNG 是免费的）.
    • Online Logo Maker - 在线制作标志. 额外的用户帐户功能来保存创作.
    • Coolors - 调色板生成器.
    • Tiny PNG - 智能 PNG 和 JPEG 压缩.
    • Realtime Board [Account] - 大面积、屏幕共享、视频和语音通信的协作黑板.
    • ezGIF - GIF 图像编辑器（裁剪、缩放、优化等）.
    • Adobe Color CC - 配色方案设计师.
    • Burner Bonanza [Account] - 删除图像背景并立即在 Facebook 或 Twitter 上分享的工具，登录后可将编辑后的图像下载为 PNG 或 JPEG.
    • IcoMoon - 一个平台，可帮助您查找矢量图标包，将图标托管为 SVG 或字体，并生成所选图标的 SVG + PNG + 图标字体.
