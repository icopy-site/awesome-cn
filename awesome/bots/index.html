    Tools For Building Bots¶

    • wit.ai - 轻松创建人类可以与之聊天的基于文本或语音的机器人.
    • Botkit - Botkit 简化了设计和运行 Slack 内部机器人的过程.
    • Dialogflow - 建立自然而丰富的对话体验.
    • Text It - 可视化构建短信和语音应用程序以吸引您的客户.
    • Chatfuel - 具有 AI 导航功能的直观机器人构建器. 无需编码.
    • Watson - 为您的应用程序带来认知计算的力量.
    • Beep Boop - Beep Boop 是一个简单的托管平台，适用于您的 Slack 和 Messenger 机器人.
    • Dexter - 获取一个 Slack 机器人，它会根据您自己的 Google 表格中的答案做出响应.
    • Converse AI - 使用智能聊天机器人技术即时回复客户问题.
    • Gupshup - 在任何渠道上构建和部署机器人的最简单和最快的方式.
    • Haven OnDemand - 它们是与 watson 类似的 API，但有据可查且有免费增值版本.
    • Recast.AI - 面向开发人员的协作机器人平台：轻松构建对话机器人.
    • ChatterBot - 以 Python 构建的基于机器学习的会话对话引擎
    • Google Cloud Platform - 使用文本块的情感分析来为对话机器人添加决策树逻辑.
    • Messenger Demo Viewer - 在不显示个人聊天记录的情况下演示 FB Messenger 机器人
    • BotStar - 通过智能训练直观地设计和开发聊天机器人的强大平台.

    Tools For Bot Analytics¶

    • ChatMetrics - 机器人的参与平台. ChatMetrics 允许重新激活不活跃的用户并通过向用户发送“智能”通知来提高保留率.
    • Dialog - 供营销人员使用的聊天机器人分析平台，可推动用户参与、保留、转化和了解行为.
    • Dashbot - 可操作的机器人分析：增加用户参与度、获取和货币化.
    • Botmetrics - 发展并留住您的聊天机器人客户. 企业级开源对话分析、营销自动化和 CRM 集成到单个代码片段中.

    Tools For Bot Conversation Mockups¶

    • Bot Society - 用于演示和演示的机器人对话模型.
    • Bot Sketch - 素描聊天机器人 UI（仅限 Mac）
    • Botframe - 设计机器人的游乐场（对话模型）.
    • Botmock - 从一个简单的拖放编辑器创建对话流和交互式原型，然后共享或测试这些原型并在一个集中位置获得反馈.
    • BotStar - 一个成熟的聊天机器人平台，但可视化编辑器也可以用作模型工具.

    General¶

    • Qtypes - Node.js 中基于规则的答案类型分类系统.
    • BotKit - Botkit 是用于制作机器人应用程序的工具包.
    • Claudia Bot Builder - 为 FB、Slack、Skype 和 Telegram 创建聊天机器人，并在几分钟内部署到 AWS Lambda.
    • Universal Bot Framework - 仅使用一种逻辑为 FB Messenger、Kik、Telegram 和 Skype 创建跨平台聊天机器人.
    • Pandorabots - 用于构建和部署聊天机器人的网络服务.
    • Microsoft Bot Framework - 用于构建、连接、测试和部署功能强大的智能机器人的 Node.js/.NET 库.
    • Bottender - 以您的方式快速灵活地制作机器人.
    • BotMan - PHP Bot Framework 支持十几个平台（FB、Instagram、MS Bot Framework 等）
