    Awesome ibmcloud

    Awesome IBM Cloud¶

    精选的 IBM Cloud SDK、开源存储库、工具、博客和其他资源的精选列表.

    Carbon Design System¶

    使用 IBM 构建的设计系统的附加组件、组件、图标和其他资源.

    • carbon - IBM 构建的设计系统.
    • carbon-addons-angular - Carbon Angular 项目的通用插件.
    • carbon-addons-ics - 用于 IBM 协作解决方案的 Carbon 插件.
    • carbon-addons-iot-react - 该库包含在所有 IBM IoT 产品之间共享的 React 组件，以统一我们的外观. 这些小部件通常基于 carbon-components-react 小部件.
    • carbon-angular-starter - 一个简单的入门应用程序，用于使用 Carbon 组件引导应用程序.
    • carbon-badges - 申请碳徽章的网站.
    • carbon-charts - 使用 D3 和打字稿实现的可重用图表.
    • carbon-components-angular - IBM Carbon 设计系统的 Angular 实现.
    • carbon-components-react - 碳设计系统的反应组件.
    • carbon-components-svelte - 碳设计系统的巧妙实施.
    • carbon-components-vue - Carbon 设计系统的 Vue 实施.
    • carbon-design-kit - Carbon Design System 视觉资产（组件、图标、调色板、网格、模板）的版本化综合套件.
    • carbon-icons - Carbon 设计系统的 SVG 图标库.
    • carbon-icons-angular - Angular 中的碳图标！
    • carbon-icons-svelte - 使用 Carbon Design System 的数字和软件产品图标的 Svelte 组件.
    • carbon-language-website - IBM 设计语言网站.
    • carbon-sandbox - Carbon Components 的主题游乐场.
    • carbon-tutorial - Carbon Design System 教程的应用程序.
    • carbon-web-components - Web 组件之上的 Carbon Design System 变体.
    • carbon-website - 碳设计系统的网站.
    • gatsby-starter-carbon-theme - 演示 Gatsby Carbon 主题的 Gatsby 启动器.
    • gatsby-theme-carbon - 以 Carbon 为灵感的 Gatsby 主题.
    • ibm-dotcom-library - 基于 IBM Carbon 设计系统的 IBM.com 库.
    • ibm-dotcom-library-nextjs-template - 这是一个利用 IBMdotcom React 库的 IBMdotcom NextJS 模板.
    • ibm-dotcom-library-website - 这是 IBM.com 图书馆网站，其中包括有关 IBM.com 设计和开发的文档和指南.
    • ibm-security - 由 IBM Security 构建的 Carbon 体验框架.

    Command Line Tools¶

    适用于 MacOS、Windows 和 Linux 的命令行工具.

    • Cloud Foundry CLI - Cloud Foundry 的官方命令行客户端.
    • The Kui Framework for Graphical Terminals - 用于云原生开发的混合命令行/UI 开发体验.
    • IBM Cloud CLI Release - 这是 IBM Cloud 的命令行客户端.
    • IBM Cloud Docker Containers - *非官方- 不同 Docker 容器的集合，用于与 IBM Cloud 交互.
    • Universal Command Line Interface - IBM Cloud CLI 提供命令行界面用于管理 IBM Cloud 中的资源.
    • Windows 64-bit - 适用于 64 位 Windows 的命令行界面.

    Infrastructure¶

    使用 Softlayer 部署和管理您的应用程序.

    • cloudland - Cloudland 是一个轻量级基础架构即服务项目，加上 OpenShift 4 平台即服务部署引擎，是一个用于管理 VM 实例、软件定义网络 (SDN) 和卷的系统框架.
    • cp4d-deployment - 这些 terraform 脚本被开发用于在公共云上部署 CloudPakforData.
    • ibm-spectrum-scale-cloud-install - IBM Spectrum Scale 的云资源配置框架.
    • lamp-terraform-ibm - 使用基础架构即代码自动部署环境.
    • terraform-provider-ibm - Terraform IBM Cloud 提供商.
    • vision-terraform - 用于在 IBM Cloud 中部署的 IBM Visual Insights 示例地形模板（以前称为 IBM PowerAI Vision）.

    Platform¶

    使用 Cloud Foundry 部署和管理您的应用程序.

    • charts - IBM/charts 存储库为 IBM 和第三方中间件提供了 helm 图表.
    • cloud-foundry - IBM 云平台即服务.
    • binary-buildpack - 将二进制文件部署到 Cloud Foundry.
    • dotnet-buildpack - Linux 上 .NET Core 的 Cloud Foundry 构建包.
    • go-buildpack - 用于 Go 语言的 Cloud Foundry buildpack.
    • ibm-websphere-liberty-buildpack - IBM WebSphere Application Server Liberty Buildpack.
    • java-buildpack - 用于运行 Java 应用程序的 Cloud Foundry buildpack.
    • nginx-buildpack - 提供 NGINX 的 Cloud Foundry buildpack.
    • nodejs-buildpack - 用于 Node.js 的 Cloud Foundry buildpack.
    • php-buildpack - 用于 PHP 的 Cloud Foundry 构建包.
    • python-buildpack - Python 语言的 Cloud Foundry buildpack.
    • r-buildpack - 用于 R 的 Cloud Foundry buildpack.
    • ruby-buildpack - 用于 Ruby、Sinatra 和 Rails 的 Cloud Foundry buildpack.
    • staticfile-buildpack - 将静态 HTML/JS/CSS 应用程序部署到 Cloud Foundry.
    • cloud-operators - IBM 公共云服务运营商.
    • cloud-pak - IBM Cloud Paks 是企业级容器化软件，将容器镜像与企业功能相结合，用于在生产用例中进行部署，并集成了管理和生命周期操作. 基于产品专业知识的预配置部署、滚动升级、回滚、安全/漏洞测试等功能.
    • generator-swiftserver - Kitura REST 网络服务服务器的生成器.
    • go-cloud - 用于对托管 Go 应用程序的云平台进行基准测试的 Golang 样板代码.
    • go-website - 使用 Golang 服务器部署您的网站.
    • ibm-cloud-go - 用于访问 IBM Cloud API 的 Go 库.
    • insurance-bot - 基于消息的保险体验.
    • kitura - A Swift web framework and HTTP server.
    • kitura-couchdb - 用于 Kitura 的 CouchDB 适配器.
    • kitura-credentials - A pluggable framework for validating user credentials in a Swift server using Kitura.
    • kitura-credentials-google - Kitura-Credentials 框架的插件，使用 Google 网络登录进行身份验证.
    • kitura-net - Kitura 网络.
    • kitura-nio - 基于 SwiftNIO 的 Kitura 网络库.
    • kitura-sample - 一个示例应用程序，展示了如何使用 Kitura 的各种功能.
    • kitura-session - 使用 Kitura 在 Swift 服务器中管理用户会话的可插入框架.
    • kitura-starter - 用于 IBM Cloud 的基于 Kitura 的服务器启动器应用程序.
    • logistics-wizard - Logistics Wizard 是一种端到端的智能供应链管理解决方案.
    • multiple-deployment-options - 显示如何将一项服务部署到多个部署选项.
    • node-mqtt-for-anki-overdrive - Node.js Controller and MQTT API for Anki Overdrive.
    • swift-ubuntu-docker - 与 Swift 二进制文件打包在一起的 Docker 图像.
    • swifty-request - SwiftyRequest 是一个为 Swift 构建的 HTTP 网络库.
    • watson-assistant-variables - 在 IBM Watson Assistant（以前称为 Conversation）服务中使用上下文变量和（系统）实体的样本.
    • webiste-base - 使用 Node.js 服务器部署您的网站.

    Serverless¶

    使用 IBM Cloud Functions（基于 IBM Cloud 的 Apache OpenWhisk）部署和管理您的操作/功能.

    • openwhisk-darkvisionapp - 使用 IBM Watson 和 IBM Cloud Functions 发现视频中的暗数据.
    • openwhisk-visionapp - 使用 IBM Cloud Functions 构建的用于图像标记和面部检测的示例 iOS 应用程序.
    • serverless-chatbot - 使用无服务器服务（无服务器架构/功能即服务）构建您自己的聊天机器人.
