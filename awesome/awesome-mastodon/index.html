    Mastodon

    Awesome Mastodon ¶

    Mastodon is the world's largest free, open-source, decentralized microblogging network.

    This list is a collection of the best Mastodon resources.

    Official¶

    • GitHub Repository
    • Documentation & Help Center
    • API Documentation

    Instances¶

    • Official Instance List - Official List of Mastodon instances.
    • mastodonservers.net Instance List - Larger list of Mastodon instances.
    • instances.social Instance List - Autogenerated list of Mastodon instances.
    • mastodon.social - The largest Mastodon server with over 333k active users and 1.5m registered users.
    • EU Server - The official server for EU institutions.
    • Vivaldi Social - Mastodon social hosted by Vivaldi.
    • Mozilla Social - Mastodon social hosted by Mozilla.
    • Journa.host - Mastodon server for journalists.
    • Newsie - Another Mastodon server for journalists.
    • Bots in Space - Mastodon server for bot accounts.

    Statistics¶

    • The Federation - Statistics on protocols, users, and nodes of the Fediverse.
    • FediDB - Fediverse network statistics.
    • FediBuzz - Observe trends across the Fediverse.
    • Fediverse Graph Visualization - Graph visualization of all complete Fediverse.
    • MastoInstance.Info - View instance metadata and explore the federation graph.
    • fediverse.party - Detailed information about the most popular Fediverse projects and protocols.

    People¶

    • Academics On Mastodon - Markdown list of academics on Mastodon, by discipline/group/area.
    • Spreadsheet of Popular accounts - Google spreadsheet of the most popular Mastodon accounts of the Fediverse.
    • Spreadsheet of Journalist accounts - Google spreadsheet of various journalists of the Fediverse.

    Tools¶

    • Toot Scheduler - Schedule toots to "toot" later.
    • Mastodon List Manager - Create, manager, filter, and search lists of your selected Mastodon users.
    • Mastodon Toot Bookmarklet - Bookmarklet to toot the current page.
    • Mastodon - Simplified Federation! - Redirect clicks on remote follow/interaction buttons to your own instance.
    • Mastodon Link - Adds Mastodon icon links by profile to make following people on other instances easier.
    • Mastodon Redirector - Adds a button to view profiles from other instances on your profile page.
    • mastotool - Collection of command-line tools for working with Mastodon accounts.
    • Mastodon Widgets - JavaScript Web Components to embed a Mastodon profile, timeline, follow and share button into any website. Provides an auto-completing instance selector.

    Desktop Clients¶

    • Official Client List - Official list of Mastodon clients.
    • Elk - A nimble Mastodon web client.
    • Brutaldon - A brutalist web interface for Mastodon.
    • phanpy - A minimalistic and opinionated Mastodon web client.
    • Planiverse - Minimalist, no-JS Web client for Mastodon.
    • Official Client - Official Android client.
    • Whalebird - Electron-based Mastodon client.
    • toot - Mastodon CLI & TUI in Python.
    • Tokodon - Mastodon client made by KDE.
    • hellclient - Mastodon CLI readline client written in Go.

    Mobile Clients¶

    • Megalodon - Fork of the official Android client.
    • Fedilab - Fully-featured Android client.
    • Tusky - Intuitive Android client.
    • Ivory - Mastodon client for iOS, by the creators of Tweetbot.
    • Toot! - Mastodon client for iOS.
    • SwiftUI - Mastodon client for iOS using SwiftUI.

    Libraries¶

    • Official Library List - Official list of Mastodon client libraries.
    • masto.js - Mastodon client for JavaScript.
    • go-mastodon - Mastodon Client for golang.

    Servers¶

    • Glitch Edition - Mastodon Fork with local posting, toot formatting, improved settings, and more granulate controls.
    • Hometown - Mastodon Fork with local posting, following lists, customizable max toos length.
    • ecko - Mastodon fork with local posting, toot formatting, and more granulate controls.

    Guides¶

    • What is Mastodon? (Official Video) - What is Mastodon? (Official Video).
    • A Beginner's Guide to Mastodon - A Beginner's Guide to Mastodon.
    • Fedi.Tips - An unofficial guide to Mastodon and the Fediverse.

    Hosting¶

    • Official Hosting Documentation - Official page with introduction and list of hosting providers.
    • Masto.host - Fully managed Mastodon hosting.
    • toot.io - Managed Mastodon Hosting, used by ACM, Microsoft Dynamics, etc.
    • Cloudplane - Fully manged hosting for open-source apps.
    • elestio - Fully-managed Mastodon hosting.

    Similar Projects¶

    • aether - Peer-to-peer ephemeral public communications.
    • Pleroma - Lightweight microblogging platform.
    • GNU social - Oldest microblogging platform.
    • Microblog.pub - Single-user lightweight microblogging platform.
    • Hubzilla - Blog/social networks platform with file, contacts and events sharing.
    • Friendica - Social network platform.
    • Peertube - Video sharing platform.
    • FunkWhale - Audio sharing platform.
    • Plume - Blogging platform.
    • WriteFreely - Blogging platform.
    • PixelFed - Photograph sharing platform.
    • Misskey Hub - Interplanetary microblogging platform.
    • Lemmy - Link aggregation platform.
    • Kbin - Link aggregation platform.

    Bots¶

    • feed2toot - Automatically parses RSS feeds, identifies new posts and posts them on Mastodon (Python).
    • aerialbot - Respond with geotagged aerial imagery of a random location in the world.
    • rust-trending - Post trending Rust repositories on Mastodon (inspired by TrendingGithub).
    • @TrendingBot@mastodon.social - Shows you what's trending on Mastodon.
    • News Bot - Mirrors Twitter accounts on Mastodon (ClojureScript), source available on GitHub.
    • @HackerNewsBot@mastodon.social - Post Hacker News posts with over 100 upvotes.
    • @launchradar@mastodon.cloud - News about space flight, astronomy and astrophysics.
