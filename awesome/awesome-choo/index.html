    choo

    Awesome choo
    ¶

    choo 是一个用于创建的 4kb 框架 > 坚固的前端应用程序

    Official resources¶

    • Docs
    • Handbook
    • Repo
    • Website
    • Twitter thread

    Dependencies¶

    choo 是一个模块化框架. 这些是它粘合在一起的依赖项 在引擎盖下：

    • bel - 使用创建可组合的 DOM 元素 模板字符串.
    • hyperx - 将模板字符串转换为 库后端.
    • nanomorph - 真实 DOM 节点的超快速差异算法.
    • nanoraf - 仅在需要时致电 RAF.
    • nanorouter - Smol 前端路由器.
    • nanobus - 微型消息总线.
    • nanolocation - 小 window.location 库.
    • nanohref - 微型 href 点击处理程序库.
    • nanoquery - 微小的查询字符串模块.
    • nanotiming - 小型时序库.

    Plugins and addons¶

    • choo-location-electron - 在电子中修复 choo 的路由器.
    • choo-log - choo 的开发记录器.
    • choo-test - Easy choo 应用程序单元测试.
    • choo-persist - 将 choo 状态与 LocalStorage 同步.
    • choo-promise - 在效果和订阅中使用承诺.
    • choo-pull - 包装处理程序以在 choo 插件中使用拉流.
    • choo-redirect - 将一个视图重定向到另一个视图.
    • choo-model - choo 的实验状态管理库.
    • choo-resume - choo-resume + hot-rld = choo 中的热应用重新加载.
    • choo-detached - 使用 choo 作为可安装的、简单的独立组件（无路由）.
    • choo-service-worker - choo 的服务工作者加载程序.
    • choo-websocket - 围绕 WebSocket 浏览器 API 的小型包装器，用于“choo”应用程序.
    • choo-store - 用于 choo 应用程序的轻量级状态结构.

    Elements¶

    • dom-notifications - 受 Atom 启发的通知组件.
    • choodown - 用于 choo 的简单降价组件.
    • choo-md-editor - 可在 Choo 应用程序内使用或作为独立库使用的轻量级降价编辑器.
    • choo-chartist - 一个使用的小组件 Chartist 使用 choo 框架.
