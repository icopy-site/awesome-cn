    Community¶

    • Craft Coding Challenge - Craft CMS 开发人员测试和提高编码技能的培训基地.
    • Craft Cookbook - 常见 Craft CMS 问题的快速解决方案.
    • CraftQuest - 无限制地参加工艺培训（以及更多）.
    • Discord - Craft CMS 的官方 Discord 社区.
    • Facebook - 喜欢官方 Craft CMS Facebook 页面.
    • Issues, Feedback, Discussions - 在 GitHub 上发布功能想法和错误报告.
    • Meetups - 寻找当地的 Craft 聚会.
    • StackExchange - 针对管理员、最终用户、开发人员和设计人员的问答.
    • Twitter - 关注对话，官方社区标签是 #craftcms.
    • Mastodon - 最新消息，在最新的网络上.
    • Work With Craft - 专门的工作委员会旨在将雇主与工艺专业人士联系起来.
