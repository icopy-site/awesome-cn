    Awesome waves

    Awesome Waves ¶

    Waves 是用于 Web 3.0 应用程序和去中心化解决方案的开源区块链平台和开发工具集.

    此列表是在 Waves 区块链上创建去中心化应用程序的有用资源的集合.

    Official¶

    • Official website
    • Blog on Medium
    • Waves on GitHub
    • Documentation

    Playground¶

    Testnet¶

    Testnet 是一个稳定的网络，并从主网完全复制了配置和节点版本，但允许获得免费的 Waves 代币来测试你需要的东西.

    • Explorer - 官方测试网浏览器. 它有一个 faucet 获得一些免费的 Waves 代币.
    • Node API - 测试网节点的 API.
    • Data Service - 测试网的官方数据服务.

    Stagenet¶

    Stagenet 是一个不稳定的网络，用于在将 Waves 官方产品发布到主网和测试网之前对其进行测试. 但您也可以赚取免费的 Waves 代币来测试您需要的东西. 例如，您的服务或 DApp 与即将发布的版本的兼容性.

    • Explorer - 官方探险家. 它有一个 faucet 获得一些免费的 Waves 代币.
    • Node API - Waves 节点的 Swagger REST API.
    • IDE - 用于在 Stagenet 上创建 Ride 智能合约的在线 IDE.

    Docker images¶

    • WavesNode - Waves 平台节点的 Docker 镜像. 支持任何官方或自定义网络.
    • K8s by loxal - 构建容器是为了在 Kubernetes 集群中作为即发即弃的“StatefulSet”运行.
    • waves-private-node - 私有本地 Waves 节点. 运行您自己的隔离节点以开发智能合约和任何实验的最简单方法.
    • explorer - 用于您的节点的 Waves Explorer.

    Frameworks and tools¶

    The Ride programming language¶

    • Waves IDE - 用于在 Ride 上创建智能合约的在线 IDE.
    • ride-examples - 使用 Ride 的帐户和资产脚本示例.
    • waves-ride - 支持 Ride 的 Visual Studio Code 扩展.
    • vim-ride - 用于 Ride 语法突出显示的 Vim 插件.
    • surfboard - 使用 Ride 语言和测试的 CLI.
    • Paddle - 为您的 dApp 和其他智能合约编写测试的 Java 库.

    Client libraries¶

    Go¶

    • go-lib-crypto - 用于 Waves 的统一加密原语的 GoLang 实现.

    PHP¶

    Python¶

    Rust¶

    • WavesRS - Waves 区块链的 Rust 接口.
    • waves-lib-rust - 与 Waves 区块链一起工作的库.

    Swift¶

    • WavesSDK-iOS - 适用于 iOS 上移动应用程序的 SDK.

    Blockchain utils¶

    • One Million Pixel Waves - 智能合约支持的网站完全运行在 Waves 区块链上，任何人都可以拥有一块区块链并通过图像和链接使其永生.
    • Signature Chain - Signature Chain 的主要目标是开发一个允许文件和文档认证的平台.
    • Auctionlance Platform - 自由职业者市场，客户可以在其中使用 Waves、Waves 代币和其他加密货币雇佣和支付自由职业者.
