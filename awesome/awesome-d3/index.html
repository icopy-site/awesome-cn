    D3

    Awesome D3 ¶

    此列表跟踪有趣 D3js 库、插件和实用程序.
    我们决定不在这里列出教程、资源或概念，因为已经有一个 very good list of readings 在那里你可以检查并做出贡献.

    Curators: Moritz Klack 和 Christopher Möller webkid.io

    您还可以使用我们的交互式浏览列表 D3 Discovery 工具.

    Charts¶

    • billboard.js - 可重复使用的图表库 [条形图、线形图、区域图、环形图、饼图、阶梯图、样条图]
    • britecharts - 客户端可重用图表库 [条形图、线形图、圆环图、迷你图、阶梯图]
    • chart-parts - 一个灵活的、反应友好的、用于数据可视化的图形语法
    • cubism - Time series visualization
    • c3 - 可重复使用的图表库 [折线图、样条曲线、阶梯图、面积图、堆叠图、条形图、饼图、圆环图]
    • dagre-d3 - 在客户端布局有向图
    • dc.js - 对于大量数据
    • d2b - 轴图、饼图、桑基图、旭日图的图表库
    • d3-bar - 条形图
    • d3-beeswarm - 计算蜂群排列的插件
    • d3-boxplot - d3js 箱线图插件
    • d3-bumps-chart - 用于可视化颠簸图表的插件
    • d3-dag - 用于可视化有向无环图的布局算法
    • d3-ez - 易于重复使用的 D3 图表和组件 [条形图、圆环图、饼图、气泡图、玫瑰图、热图]
    • d3-flame-graph - 来自分层数据的火焰图
    • d3-funnel - 漏斗图和金字塔图表库
    • d3-gridding - 使用网格的图表模型
    • d3-heatmap - 热图
    • d3-message-sequence - 动态/静态消息序列图
    • d3-upset - 绘制 UpSet 图的函数 [upset, bar, intersections, venn, relationships]
    • d3-x3d - 使用 D3 和 X3D 的数据驱动 3D 图表 [气泡、条形图、表面积、散点图、面积]
    • d3fc - 交互式图表组件的集合 [折线图、条形图、堆积图、散点图、烛台图、ohlc]
    • d3plus - 用于轻松创建可视化效果的扩展库 [散点图、堆叠图、线图、条形图、饼图、网络图、气泡图、框图、地图]
    • d3panels - 带有链接刷牙的交互式图表[点、散点、线、热图、直方图]
    • d3pie - 可配置的饼图库和生成器
    • D3xter - 直接绘图 [情节、时间线、条形图、直方图、饼图]
    • D4 - 可重复使用的图表 DSL [条形图、环形图、折线图、散点图、堆叠图、瀑布图]
    • dTree - 家谱库
    • EventDrops - 基于时间/事件系列的交互式可视化
    • metrics-graphics - 针对时间序列数据的可视化进行了优化 [line, scatter, area]
    • micropolar - 一个极地海图库
    • mpld3 - 导出 matplotlib 图形以在浏览器中工作
    • neo4jd3 - Neo4j 图形可视化
    • nvd3 - 可重复使用的图表和图表组件 [箱形图、对接图、烛台、折线图、条形图、饼图、散点图、迷你图]
    • oecd-simple-charts - 简单的图表库[箱形图、堆积条形图、珍珠图]
    • plotly.js - 高级图表库 [散点图、折线图、条形图、饼图、箱线图、直方图、热图]
    • plottable - 灵活的交互式网络图表 [区域、条形图、折线图、饼图、散点图、堆叠图]
    • rickshaw - 用于创建交互式实时图形的工具包 [折线图、散点图、条形图]
    • taucharts - 专注于设计和灵活性的图表 [折线图、条形图、面积图、堆叠图]
    • vega - 可视化语法
    • vega-lite - 交互式图形的高级语法
    • vega-lite-api - Vega-Lite 的 JavaScript API.
    • venn.js - 面积比例维恩图和欧拉图
    • visavail - 时间数据可用性可视化
    • vizabi - 构建可视化数据探索工具的框架[bubble, map, line, bar, sankey, donut]
    • WebCola - 图形可视化和探索的布局
    • xkcdgraphs - Xkcd 样式图 [线]
    Third Party¶
    • Layer Cake - 一个建立在 Svelte 之上的图形框架.
    • nivo - 用于具有同构能力的 React 的 Dataviz 组件 [条形图、线形图、区域图、气泡图、弦图、热图]
    • ngx-charts - Angular [条形图、饼图、折线图、面积图、极坐标图、堆积图、气泡图] 的图表框架
    • potion - 用于组合可视化的 React 组件集合
    • react-d3-components - D3 组件 [条形图、堆积图、散点图、线形图、面积图、饼图]
    • react-d3-library - 在 React [area, bar, line, pie, scatter] 中使用 D3 的库
    • react-stockcharts - 高度可定制的股票图表 [面积图、折线图、散点图、气泡图、条形图、堆叠图、烛台图]
    • react-vis - 一组用于呈现可视化效果的反应组件 [区域、条形图、热图、线、散点图]
    • reaviz - 基于D3的React数据可视化库
    • recharts - 重新设计的图表库，使用 React 构建
    • semiotic - 结合React & D3的数据可视化框架[散点、线、面积]
    • victory - 用于构建可视化的可组合 React 组件 [区域、条形图、烛台、饼图、线、散点图、voronoi]
    • vx - React 的可视化组件 [bar, line, scatter, stacked, area, pattern, heatmap, pie, radial, map, geo]
    • Vs - Vue.js 的声明式可视化组件 [bar、line、area、pie、donut、stacked、sankey、timeline]
    • v-chart-plugin - 轻松将图表绑定到存储在 Vue.js 组件中的数据.

    Maps¶

    • d3-composite-projections - 一起显示国家遥远土地的预测
    • datamaps - 一个文件中的可自定义地图可视化
    • d3-topogram - 基于 TopoJSON 的连续区域地图
    • d3-exploder - 让您轻松移动和调整地理特征
    • d3-geo-polygon - 球形多边形的裁剪和几何操作
    • d3-geo-projection - 扩展的地理预测
    • d3-geo-scale-bar - 显示投影地理空间数据的自动比例尺.
    • d3-geo-voronoi - 球体的 Voronoi 图和 Delaunay 三角剖分
    • d3-geomap - 用于创建地理地图的库
    • d3.geo2rect - 将 geojson 多边形变形为矩形
    • d3-inertia - d3-drag 的扩展，以一定的惯性继续鼠标移动
    • mapmap.js - 用于交互式专题地图的数据驱动 API
    • maptable - 将数据集转换为一组可视化组件（地图、过滤器、表格）
    • simple-map-d3 - 简单的等值线样式地图
    • spam - 使用 D3 和 Canvas 轻松创建地图
    Third Party¶
    • leaflet-d3 - 将 D3 与 Leaflet [map, geo] 一起使用的插件集合
    • Wikimaps-D3js Atlas - 用于生成光栅、topojson 和 svg 地图的 CLI [map, geo]
    • react-simple-maps - React 的 SVG 映射组件库，建立在 d3-geo [map, geo] 之上

    • d3-hsluv - 人性化的 HSL（色调、饱和度、亮度）色彩空间.

    • d3-interpolate-path - 插入具有不同点数的路径
    • d3-jetpack - 漂亮的便利包装器，可加快您的日常工作
    • d3-lasso - 通过在对象上方或周围画一条线来标记元素

    • d3-scale-cluster - 将数据聚类成离散组的规模
    • d3-sparql - 用于从 SPARQL 端点访问数据的实用程序
    • d3-template - 使用 D3 的数据绑定机制创建和呈现模板的插件
    • d3-tooltip - 带有阴影和动态内容的箭头形工具提示
    • d3-voronoi-treemap - 根据 Voronoi 镶嵌计算树图
    • d3-xray - 记录数据结果的书签在您将鼠标悬停时加入
    • swoopyarrows - 用于创建俯冲箭头的插件
    • swoopy-drag - d3 图形的手工标签放置
    • textures - 用于数据可视化的 SVG 模式
